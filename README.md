## Environment Setup  

### Windows
1. Set up the virtual environment  
\> pip install python  
\> pip install virtualenv  
\> python -m virtualenv env  
\> env\Scripts\activate  

3. In (env):  
\> pip install flask  
\> set FLASK_APP=app.py  
To run the app:  
\> python flaskblog.py  
To close the virtual environment:  
\> deactivate
